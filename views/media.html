Manage Assets

As a photographer(artist/videographer), I can submit a photo(image/video).

As a photographer(artist/videographer), I can edit a photo(image/video) I submitted before it is published.

As a photographer(artist/videographer), I can delete a photo(image/video) I submitted before it is published.

As an editor, I can edit any photo(image/video) submitted.

As an editor, I can approve/publish any photo(image/video).

As an editor, I can reject a photo(image/video) (which notifies reporter to fix or delete).
